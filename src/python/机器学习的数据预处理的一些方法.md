4. 无量纲化

   - 线性的无量纲化一般包括`中心化处理` 和`缩放处理`
   - 中心化的本质：让所有记录减去一个固定值，让样本数据的均值调整为 0（例如让样本数据都减去均值）
   - 标准化：目的是将数据调整到统一尺度。通常通过使特征的均值为 0，方差为 1，或者通过缩放到一定范围（例如[0,1]）来实现
     - Z-score 标准化，将特征的均值调整为 0，方差为 1.会将数据准转换为服从标准正态分布
     - 最小-最大缩放，将数据缩放到[0，1]或者自定义的范围，适合在数据分布无特殊形状时使用

   ```python
   import numpy as np
   from sklearn.preprocessing import MinMaxScaler, StandardScaler
   
   x = np.array([[-1, 2], [-0.5, 6], [0, 10], [1, 18]])
   print(x)
   
   # 最小-最大缩放
   scaler = MinMaxScaler()
   
   result = scaler.fit(x).transform(x)
   # or
   # scaler.fit_transform(x)
   
   # 将数据还原
   scaler.inverse_transform(result)
   ```

   ```python
   # z-score标准化
   scaler = StandardScaler()
   result = scaler.fit_transform(x)
   print(result.mean())
   print(result.std())
   result
   ```

5. 处理分类数据

   很多算法都只能处理数值类型的数据（例如逻辑回归，svm，k-means）,因此我们需要对数据进行编码，**将文字型数据转化为数值型**

   - 例如[“男”，“女”] 转为 0，1
   - [支付宝，微信，现金] 转为 0，1，2
   - 上面这种编码方式称为整数编码，适用于原本数据有序的情况，很显然性别之间是没有大小的关联的，支付方式之间也没有顺序，像这样的数据比较适用于**onehot 编码**

   1. 使用整数编码对性别进行编码

   ```python
   df6 = pd.DataFrame({
     '姓名': ['John', 'Alice', 'Bob', 'Alice', 'John'],
     '年龄': [25, 30, 35, 30, 40],
     '性别': ['M', 'F', 'M', 'F', 'M']
   })
   # 将性别进行编码
   from sklearn.preprocessing import OrdinalEncoder
   
   x = df6["性别"].values.reshape(-1, 1)
   df6["性别"] = OrdinalEncoder(categories="auto", dtype=np.int8).fit_transform(x)
   df6
   ```

   2. 使用 onehot 编码对性别进行编码

      ```python
      from sklearn.preprocessing import OneHotEncoder
      
      x = df6["性别"].values.reshape(-1, 1)
      encoder = OneHotEncoder(categories="auto", dtype=np.int8)
      
      new_data = pd.DataFrame(encoder.fit_transform(x).toarray())
      
      df6 = pd.concat([df6, new_data], axis=1)
      df6.drop(columns=["性别"], inplace=True)
      
      # 对列重命名
      df6 = df6.rename(columns={0: "女", 1: "男"})
      df6
      ```

6. 处理连续型特征：二值化与分段

   - 根据阈值将数据二值化（0，1），例如年龄大于 20 映射为 1，小于 20 映射为 0（可以决定仅考虑某种现象的存在与否）

   - 分段：将连续型变量划分为分类变量的类，例如 0-20 为一类，20-40 为一类……

     ```python
     from sklearn.preprocessing import KBinsDiscretizer
     
     X = df6["年龄"].values.reshape(-1, 1)
     enc = KBinsDiscretizer(3, strategy="uniform")
     enc.fit_transform(X)
     ```
