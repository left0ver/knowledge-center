1. 下载 dataset 的时候只需要一个进程下载即可
2. 在分布式训练中，每个 GPU 会计算他们对应的均值和方差，而 BN 的归一化和 batch size 密切相关，尤其是当 batch size 较小时，BN 的效果会变差，而`SyncBatchNorm`可以在计算 BN 的时候执行 all-gather 操作，得到所有 GPU 上的均值和方差，并计算全局的均值和方差，同时更新`running_mean` 和 `running_var` 来保证 BN 的行为和单卡的一致，因此在初始化 model 之后，通常会调用`nn.SyncBatchNorm.convert_sync_batchnorm` 将`BN`转为`SyncBatchNorm`

```python
    # 将 BatchNorm 转换为 SyncBatchNorm
    model = nn.SyncBatchNorm.convert_sync_batchnorm(model)
```

## 自己实现数据并行
1. 参数初始化的时候需要保证所有 GPU 上的初始化都一样，因此通常在初始化的时候会广播 rank0 的参数到别的 GPU 上
2. DP 中需要把数据分配到不同的 GPU 上，需要使用 pytorch 中提供的`DistributedSampler`进行采样
3. 计算完 loss 之后分别在不同的 rank 上进行 backward，之后需要把梯度聚合起来并分发到所有的 GPU 上，之后再每个 GPU 再分别更新参数即可
4. 保存模型的时候只需要单个 GPU 保存即可，因为每个 GPU 上的模型参数是一样的

## FSDP1
1.　需要先使用 FSDP 包裹 model 对参数等进行分片，再创建优化器

```python
    model = FSDP(
        model,
        auto_wrap_policy=size_based_auto_wrap_policy,
        # offload_params=True ，则可以实现将参数和显存卸载到CPU上 ，类似于zero-offload，如果显存比较紧张的时候可以使用这个
        cpu_offload=CPUOffload(offload_params=False),
        # 默认的分片策略为FULL_SHARD，即zero3
        # SHARD_GRAD_OP 为zero2
        # NO_SHARD不分片，退化为DDP
        # HYBRID_SHARD: 在单个节点上做FULL_SHARD （适合多机多卡）
        # _HYBRID_SHARD_ZERO2:在单个节点上做SHARD_GRAD_OP （适合多机多卡）
        sharding_strategy=ShardingStrategy.FULL_SHARD,
        # mixed_precision=MixedPrecision(
        #     # 参数
        #     param_dtype=torch.float16,
        #     # 梯度
        #     reduce_dtype=torch.float16,
        # ),
        # 
        sync_module_states=  True,
        device_id = local_rank
    )
    optimizer = optim.Adam(model.parameters(), lr=0.1)
```

2. 检查点的保存和加载通常有两种方式，分别是`FULL_STATE_DICT` 和`SHARDED_STATE_DICT`，

`FULL_STATE_DICT` 一般是将模型状态、优化器状态等 offload 到 CPU 上，再聚合到 rank0 上，再由 rank0 保存为一个文件

`SHARDED_STATE_DICT` 则是每个 GPU 分别保存自己的对应分片上的模型状态和优化器状态

**保存**

对于`FULL_STATE_DICT` ：

1. 模型的`state_dict`则直接通告`model.state_dict`即可得到
2. 而优化器的`state_dict`需要调用`FSDP.optim_state_dict(model, optimizer)`来获得

对于`SHARDED_STATE_DICT` ：

1. 可以直接使用 DCP 的 api 来保存检查点，会简单很多；通过 `model_state_dict, optimizer_state_dict = get_state_dict(model, optimizer)` 得到模型和优化器的状态，再调用`dist_checkpoint.save()`保存检查点，会将每个 rank 上的状态保存为一个文件，所有文件放在同一个目录下

```python
    if args.state_dict_type == "full":
        full_state_dict_config = FullStateDictConfig(
            offload_to_cpu=True, rank0_only=True
        )
        full_optim_config = FullOptimStateDictConfig(
            offload_to_cpu=True, rank0_only=True
        )

        with FSDP.state_dict_type(
            model,
            StateDictType.FULL_STATE_DICT,
            full_state_dict_config,
            full_optim_config,
        ):
            # 需要每个进程都调用
            model_state_dict = model.state_dict()
            optim_state_dict = FSDP.optim_state_dict(model, optimizer)
            state_dict = {
                "model": model_state_dict,
                "optimizer": optim_state_dict,
                "train_state": {
                    "epoch": num_epochs,
                },
            }
            if local_rank == 0:
                torch.save(state_dict, f"{args.model}_fsdp1_final.pth")
                print(f"keys : {state_dict.keys()}")
                print(f"模型已保存到 {args.model}_fsdp1_final.pth")
            dist.barrier()  # 确保所有进程在保存后再继续

    elif args.state_dict_type == "shard":
        model_state_dict, optimizer_state_dict = get_state_dict(model, optimizer)
        sharded_state = {
            "model": model_state_dict,
            "optimizer": optimizer_state_dict,
            "train_state": {
                "epoch": num_epochs,
            },
        }

        dist_checkpoint.save(
            sharded_state,
            storage_writer=dist_checkpoint.FileSystemWriter(
                f"sharded_{args.model}_fsdp1", overwrite=True
            ),
        )
```

**加载**

对于`<font style="background-color:rgba(0, 0, 0, 0.06);">FULL_STATE_DICT</font>` ：

1. 使用`FSDP.optim_state_dict_to_load(model, optimizer, state_dict["optimizer"])`来加载优化器的状态
2. `model.load_state_dict(state_dict["model"])`来加载 modle 的 state_dict

> 需要在每个 rank 上都运行
>

对于`<font style="background-color:rgba(0, 0, 0, 0.06);">SHARDED_STATE_DICT</font>` ：

1. 直接使用 DCP 的 api 来加载会方便的多，首先需要声明 state_dict,因为`dist_checkpoint.load()`是直接进行替换的，跟`torch.load`的行为不太一样
2. 加载完检查点之后，只需要调用`set_state_dict()`来设置 model 和 optimizer 的状态即可

```python
model = FSDP(...)
optimizer = optim.Adam(model.parameters(), lr=0.1)
if args.resume is not None:
        if args.state_dict_type == "full":
            full_state_dict_config = FullStateDictConfig(
                offload_to_cpu=True, rank0_only=True
            )
            full_optim_config = FullOptimStateDictConfig(
                offload_to_cpu=True, rank0_only=True
            )
            with FSDP.state_dict_type(
                model,
                StateDictType.FULL_STATE_DICT,
                full_state_dict_config,
                full_optim_config,
            ):


                print(f"从full的检查点{args.resume}恢复模型")
                state_dict = torch.load(
                    args.resume, weights_only=False, map_location="cpu"
                )

                model.load_state_dict(state_dict["model"])
                FSDP.optim_state_dict_to_load(model, optimizer, state_dict["optimizer"])
                print(state_dict.keys())
                print(
                    f" cur rank = {local_rank}, train_state = {state_dict['train_state']}"
                )
        # 使用DCP 进行加载
        if args.state_dict_type == "shard":
            model_state_dict, optimizer_state_dict = get_state_dict(model, optimizer)

            state_dict = {
                "model": model_state_dict,
                "optimizer": optimizer_state_dict,
                "train_state": {
                    "epoch": None,
                },
            }
            assert args.resume is not None and os.path.isdir(args.resume)
            dist_checkpoint.load(state_dict, checkpoint_id=args.resume)
            print(f"train_state = {state_dict['train_state']}")
            set_state_dict(
                model,
                optimizer,
                model_state_dict=state_dict["model"],
                optim_state_dict=state_dict["optimizer"],
            )

```
